## 🚀 Installation & Setup

### Prerequisites
- **Node.js** ≥ 16.0.0
- **npm** ≥ 8.0.0 (or yarn/pnpm)
- **Git**

### Quick Start

1. **Clone the Repository**
   ```bash
   git clone https://github.com/PavanKusunuri/pavan_kusunuri_portfolio.git
   cd pavan_kusunuri_portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install --legacy-peer-deps
   # or
   yarn install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
   > Server runs at `http://localhost:5173`

4. **Build for Production**
   ```bash
   npm run build
   # or
   yarn build
   ```

5. **Preview Production Build**
   ```bash
   npm run preview
   # or
   yarn preview
   ```

## ⚙️ Performance Optimizations

This portfolio includes several advanced performance optimizations:

### 📦 **Code Splitting & Lazy Loading**
- Heavy 3D libraries (Three.js, @react-three/fiber) are lazy-loaded
- Canvas components split into separate chunks
- Initial bundle reduced by ~40%

### 🔄 **Vendor Chunking**
```javascript
// Separate chunks for better caching
- react-vendor.js
- three-vendor.js
- motion-vendor.js
- vendor.js (other dependencies)
```

### 🎯 **Rendering Optimization**
- **Demand-based rendering** – Canvases only render when needed
- **Preload optimization** – Critical assets preloaded strategically
- **GPU memory optimization** – Removed unnecessary buffer preservation

### 🖼️ **Asset Optimization**
- Compressed 3D model files (GLTF format)
- Optimized texture atlasing
- Long-term caching headers

### 📊 **Metrics (Production)**
- **First Contentful Paint (FCP):** ~1.2s
- **Largest Contentful Paint (LCP):** ~2.1s
- **Cumulative Layout Shift (CLS):** <0.05
- **Time to Interactive (TTI):** ~3.5s
- **Bundle Size (gzipped):** ~145KB
